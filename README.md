  - 자료형 크기
    - 정수형
      - byte  = 1바이트
      - short = 2바이트 
      - int   = 4바이트
      - long  = 8바이트
    - 문자형
      - char   = 2바이트

    - 실수형
      - float  = 4바이트
      - double = 8바이트
  
    - 논리형
      - boolean=1바이트

    - int size()        : 배열에 추가된 요소 전체 개수를 반환
    - Eget(int index)   : 배열의 index 위치에 있는 요소 값을 반환
    - Eremove(int index): 배열이 index 위치에 있는 요고 값을 제거 그 값을 반환
